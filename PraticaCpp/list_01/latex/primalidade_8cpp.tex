\doxysubsection{Detailed Description}
Verifica se um número é primo. 

Esta função verifica se um número inteiro dado n é primo.


\begin{DoxyParams}{Parameters}
{\em n} & O número a ser verificado.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
true se o número for primo, false caso contrário.
\end{DoxyReturn}
\begin{DoxyNote}{Note}
A função usa um laço para verificar se o número é divisível por mais de dois números. 
\end{DoxyNote}
